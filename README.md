# 프로젝트 소개
스프링 배치 프레임워크를 사용해 <br>
영화진흥위원회의 OpenAPI를 통해 얻어온 데이터를 <br>
DB에 적재하는 토이 프로젝트입니다. <br>
<br>
<br>

# 개발 환경
Jdk                : 17 <br>
Springboot         : 3.20 <br>
IDE                : IntelliJ <br>
Framework & Library: Spring Batch, Spring Data JPA, Lombok <br>
DB                 : MariaDB, H2 <br>
